# Matching-RabbitMQ

## 개요 
- RabbitMQ를 활용하여 1:1 매칭시스템을 구현했습니다.
- 사용자들의 관심사나 위치를 기반으로 매칭을 진행하며,<br> 
  효율적인 메세지 관리와 큐 관리를 통해 높은 동시성을 지원합니다.

<br>

## 주요 기능
- 사용자 간 1:1 매칭 서비스
- 관심사 및 위치 기반 매칭
- RabbitMQ 기반 큐 관리
